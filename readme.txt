== Description ==

WP Tag Delete - Tag Assassin, is a simple yet powerful plugin to delete those unused and unwanted tags.
This is very useful if you have a auto blogging plugin making gibrish tags. Tags you dont want on your webiste.

This is a must have plugin for all people who have a auto blogging plugin.

== Installation ==

Upload the WP DeleteTags - TagAssassin to your blog (in the 'wp-content/plugins' directory) (be sure to upload the "wp-deletetags" directory or it will not work.

Activate it in the plugins menu in your wp-admin control panel
